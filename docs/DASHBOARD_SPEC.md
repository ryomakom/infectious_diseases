# ランキング型ダッシュボード 仕様

## 1. 目的・コンセプト

- **主眼**: 「いま、どの感染症が・どこで・どれくらい増えているか」をランキング形式で示す。
- 時系列グラフは補助とし、**平年比**を主指標、**警報基準比**を併記（該当する感染症のみ）する。

---

## 2. 指標の定義

### 2.1 現在値（直近の水準）

- **定義（ランキング用）**: **直近確定週**の定点あたり報告数。同週に複数日ある場合は**最も新しい日の値**を使用する（4週移動平均ではない）。
- 表示上は「○月○日時点」「当週の数値」などと明示する。

### 2.2 平年値（基準となる水準）

- **定義**: **過去3年間**の**同じ週**を中心にした「4週間の移動平均」を各年で算出し、その**3つの値の平均**を平年値とする。
  - 例: 2025年第10週の平年値 ＝  
    (2022年第10週の4週MA ＋ 2023年第10週の4週MA ＋ 2024年第10週の4週MA) ÷ 3
- 週番号は**感染症発生動向調査の週**（年・週）で揃える。日付ではなく「第○週」で比較する。
- （COVID-19流行期のマスク等で感染症が極端に減っていた時期を避けるため、過去5年ではなく3年としている。）

### 2.3 平年比（全感染症で使用）

- **式**: `平年比 ＝ 現在値 ÷ 平年値`
- 平年値が 0 の場合は算出不可とする（表示は「—」など）。
- ランキングの**主ソート指標**とする。

### 2.4 警報基準比（基準が定められている感染症のみ）

- **式**: `警報基準比 ＝ 現在値 ÷ 警報レベル開始基準値`
- 基準値は「警報レベル（開始基準値）」のみを使用（終息基準値・注意報は別用途で利用可能）。
- 警報・注意報の基準が**国により定められていない感染症**（例: COVID-19）では算出しない。ランキングには平年比のみ表示する。

---

## 3. 警報・注意報基準のソース

- **出典**: [福岡市 感染症情報（定点当たり報告数等解説）](https://www.city.fukuoka.lg.jp/hofuku/hokensho/kansensho/kansenshojoho/chosa/houkokukaisetu.html)  
  「警報・注意報レベルの基準値は、これまでの感染症発生動向調査データから定められています」とある、**国に準拠した基準**を使用する。
- **データ**: リポジトリ内 `data/alert_thresholds.csv` に格納。
  - 列: `category`, `alert_start`, `alert_end`, `attention`, `source_note`
  - 基準が存在しない感染症はこのCSVに含めない（平年比のみ表示）。

---

## 4. 表示方針（ランキング）

- **一覧形式**: **感染症 × 都道府県**のフラットな一覧（1行 ＝ 1つの感染症・1つの都道府県の組み合わせ）。
- **列の例**:
  | 感染症 | 都道府県 | 平年比 | 警報基準比 |
  |--------|----------|--------|------------|
  | インフルエンザ | 東京都 | 2.3 | 1.2 |
  | … | … | … | … |

- **平年比**: 全行に表示。デフォルトのソートは「警報基準比の降順」とし、警報に近い順が一目で分かるようにする。表の「警報基準比」「平年比」の見出しをクリックでソートできる。
- **警報基準比**: 当該感染症に警報開始基準がある場合のみ表示（ない場合は「—」または空）。
- **期間**: ランキングの「現在値」「平年値」はいずれも**直近確定週**を基準に算出する。表示上「○月○日週時点」などと明示する。
- **出典の明記**: ページ内に「警報・注意報の基準は感染症発生動向調査に基づく（出典: 福岡市感染症情報）」のような文言を記載する。

---

## 5. 実装まわり（メモ）

- **R**: 週次データ（`merged_data.csv` または raw 週次CSV）から、都道府県・感染症・週ごとに 4週移動平均を計算。平年値は過去3年同週の4週MAの平均。現在値は直近確定週の値（同週に複数日ある場合は最新日の値）。これらを組み合わせて `ranking.csv`（または JSON）を出力し、ダッシュボードで読み込む。
- **フロント**: ランキング用のテーブルを新規コンポーネントとして追加。既存の時系列グラフは「詳細を見る」用に残す構成を想定。
- **更新**: GitHub Actions で週次更新する際、ランキング用集計も実行し、`docs/results/` 等に出力する。
- **ランキングとグラフの連動**: ランキング表の行をクリック（またはキーボードで選択して Enter/Space）すると、その感染症・都道府県がグラフの選択に追加され、該当する時系列グラフまでスクロールし、そのグラフを一定時間強調表示する。強調中のグラフをクリックすると強調を手動で解除できる。
