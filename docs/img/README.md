# note ロゴについて

シェアボタンでは `note-logo.png` を参照しています。

**用意した note ロゴ（「no+e」のテキストロゴ）を使う場合**  
その画像をこのフォルダに **`note-logo.png`** という名前で保存してください。

**公式ブランドガイドラインのロゴを使う場合**  
1. [ブランドガイドライン（ロゴデータ・カラー）](https://www.help-note.com/hc/ja/articles/360000235582) を開く  
2. 「ロゴデータ・ガイドラインのダウンロード」からZIPをダウンロード  
3. ZIP内の **square** 用画像を `note-logo.png`（または SVG の場合は `note-logo.svg` にリネームして保存）で置いてください。
